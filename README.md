# ตะลุยโจทย์วรรณคดี (Thai Literature Adventure)

เว็บแอปพลิเคชันเพื่อการเรียนรู้วรรณคดีไทยในรูปแบบ Gamification โดยผสมผสานการเรียนเนื้อหา การทำแบบทดสอบ และมินิเกม Action แนว Survivor-like เข้าด้วยกันเพื่อให้การเรียนวรรณคดีเป็นเรื่องสนุกและน่าติดตาม

## ฟีเจอร์หลัก (Features)
- **Adventure Map:** แผนที่การเดินทางที่ผู้เล่นจะต้องปลดล็อกด่านวรรณคดีไปทีละเรื่อง
- **Interactive Lessons:** เนื้อหาวรรณคดีที่สรุปมาให้อ่านง่าย แบ่งเป็นหมวดหมู่ (รูปแบบคำประพันธ์, บริบททางสังคม, ข้อคิด ฯลฯ)
- **Quiz System:** ระบบแบบทดสอบเก็บคะแนนสะสม และระบบ Combo 
- **Action Minigame:** มินิเกมต่อสู้เอาชีวิตรอดที่แทรกเข้ามาหลังจบเนื้อหาแต่ละด่านเพื่อรับพลังอัปเกรด
- **Cross-platform:** รองรับการเล่นบนคอมพิวเตอร์ (คีย์บอร์ด) และสมาร์ทโฟน/แท็บเล็ต (Virtual Joystick บนหน้าจอ)

## วิธีการใช้งาน / ทดลองเล่น
สามารถเปิดไฟล์ `index.html` บนเบราว์เซอร์เพื่อเริ่มเกมได้ทันที หรือนำไปอัปโหลดขึ้นบริการโฮสติ้งฟรีอย่าง **GitHub Pages** เพื่อเปิดให้ผู้ใช้งานอื่นเข้าเล่นผ่านลิงก์ได้

## เทคโนโลยีที่ใช้
- HTML5, CSS3, JavaScript (Vanilla)
- Web Audio API (สำหรับเสียงประกอบ)
- HTML Canvas (สำหรับมินิเกมและเอฟเฟกต์ Confetti)
